# Shapeler Maker
Create your own cute Shapelers and Shapelets with the Shapeler Maker! Customize your creature's expression, color, and shape then take some photos!
<img src="./public/_README/Sprickler.png"></img>
## Features

### The Shapeler Maker incudes:
- <b>Shapeler Customization:</b>
    - Body Shape
    - Shapeler (Adult) or Shapelet (Baby)
    - Color Swatches
    - Custom RGB Slider
    - 8 Different Expressions
    - Name Customization
    - A Randomize Button
- <b>Photos:</b>
    - Countdown to photo capture
    - Cropped Images
    - Image-saving to Gallery
    - Image downloading
- <b>Shapelers</b>
    - Shapelers will look wherever your cursor is
    - Unlimited Shapeler save slots

(All saving is done through localStorage)
